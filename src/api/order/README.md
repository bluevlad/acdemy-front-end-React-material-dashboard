# Order API

Provides functions to manage orders, coupons, and payments.

## Files
- `productOrder.js`: Product order management.
- `coupon.js`: Coupon management.
- `freeOrder.js`: Free order management.

## Functions

### Product Order (`productOrder.js`)
- `getProductOrderList`
- `getProductOrderView`
- `updatePayKind`
- `updateDepositStatus`
- `insertRefund`

### Coupon (`coupon.js`)
- `getCouponList`
- `insertCoupon`

### Free Order (`freeOrder.js`)
- `getFreeOrderList`
