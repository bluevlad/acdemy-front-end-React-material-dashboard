# Components Directory

This directory contains the reusable UI components used throughout the application.

## Structure

The components are primarily custom Material UI components designed for this dashboard theme (prefixed with `MD`).

- **MDAlert**: Custom alert component.
- **MDAvatar**: Avatar display component.
- **MDBadge**: Notification badge component.
- **MDBox**: Wrapper component (likely similar to Box in MUI).
- **MDButton**: Custom button component.
- **MDInput**: Text input component.
- **MDPagination**: Pagination controls.
- **MDProgress**: Progress bar/spinner.
- **MDSnackbar**: Snackbar notifications.
- **MDTypography**: Text styling and typography component.
- **ModalPopup**: A general purpose modal popup component.
- **common**: Likely contains generic or shared functional components.

## Usage

These components should be used instead of raw HTML or basic MUI components where possible to maintain design consistency.
